# vim-editing

このプラグインは、Vimでの編集に役立つキーマッピングのコレクションを提供します。

## 機能

- カーソルの上または下に空行を挿入します。
- ビジュアルラインモードでの`I`および`A`コマンドの矩形ビジュアルモードを強制します。
- `Shift-U`をやり直し（redo）にリマップします。


## インストール

お好みのプラグインマネージャーを使用してインストールしてください。

例えば、[vim-plug](https://github.com/junegunn/vim-plug)を使用する場合：

```vim
Plug 'your-github-username/editing'
```

### 手動インストール

`editing.vim` をVimのプラグインディレクトリ（例：`~/.vim/plugin/`）にコピーします。

または、

`editing.vim` を `~/.vim/vimrc.d/`）にコピーして `~/.vimrc` に `runtime! vimrc.d/editing.vim` を追加します。

## キーマッピング

| モード   | マッピング      | 説明                            |
| ----- | ---------- | ----------------------------- |
| ノーマル  | `00`       | 現在行の下に空行を挿入します。               |
| ノーマル  | `0i`       | 現在行の下に空行を挿入し、インサートモードに入ります。   |
| ノーマル  | `<S-u>`    | やり直し（redo）。                   |
| ノーマル  | `88`       | 現在行の上に空行を挿入します。               |
| ノーマル  | `99`       | 現在行の下に空行を挿入します。               |
| ノーマル  | `9i`       | 現在のブロックの下に空行を挿入し、インサートモードに入ります。 |
| ノーマル  | `8i`       | 現在のブロックの上に空行を挿入し、インサートモードに入ります。 |
| ビジュアル | `I`        | 矩形ビジュアルモードを強制し、選択範囲の先頭に挿入します。 |
| ビジュアル | `A`        | 矩形ビジュアルモードを強制し、選択範囲の末尾に追記します。 |
